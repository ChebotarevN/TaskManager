# TaskManager
----------------
**Описание**:  Это приложение позволяет добавлять и отслеживать статус задач. Приложение может работать с файлом, базой данных.
 - **Технологический стек**: JavaFX, PostgreSQL.
 - **Статус**:  1.0.
